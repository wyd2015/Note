## .gitignore忽略规则不生效的原因：
>.gitignore只能忽略那些原来没有被追踪的文件，如果某些文件已经被纳入到版本管理中，修改此文件无效；
或者.gitignore文件是后来才加进来的，此时.gitignore也不会生效的。

## 解决办法： 
先把本地git缓存清空（改变成未被追踪的状态），然后再提交。
```yml
git rm -r --cached .
git add.
git commit -m "update .gitignore"
```
