## 一、代码编辑
```yml
alt+insert: 生成代码（get/set/构造方法）；
fori/sout/psvm+tab: for循环/输出语句/main函数自动生成；

ctrl+e: 显示最近编辑的文件列表；
ctrl+i: 实现方法；
ctrl+j: 自动补全代码；
ctrl+O: 重写方法；
ctrl+p: 方法参数提示；
ctrl+d: 复制当前行；
ctrl+x/y: 剪切/删除当前行；
ctrl+ +/-: 展开/折叠当前方法；
ctrl+shift+ +/-: 全部展开/折叠当前方法；

ctrl+alt+i: 自动缩进选中的代码段；
ctrl+alt+L: 格式化代码；
ctrl+alt+O: 自动导入类和包；
ctrl+alt+t: 生成try catch;
ctrl+alt+v: 自动完成变量声明；
ctrl+alt+space: 类或接口名自动完成；

ctrl+shift+j: 合并行；
ctrl+shift+v: 打开粘贴板，选择粘贴内容；
ctrl+shift+space: 自动补全代码；
ctrl+shift+up/down: 上下移动本行代码；

shift+F6: 重命名；
shift+enter: 向下另起一行；
shift+click: 关闭当前文件；
shift+alt+insert: 列编辑模式开启、关闭；

alt+shift+up/down: 上/下移一行
```

## 二、代码查询
```yml
F4: 查找变量来源；
alt+6： 查找TODO；
alt+7: 左侧栏显示当前文件结构；
alt+F7: 寻找选中的类、函数或者变量在项目中被引用到的地方；
alt+up/down: 在当前类中快速切换方法；

ctrl+n: 查找类；
ctrl+g: 定位行数；
ctrl+f: 在当前文件查找内容；
ctrl+r: 在当前文件替换内容；
ctrl+u: 跳转到父类；
ctrl+[/]: 跳到大括号的开头/结尾；
ctrl+w: 选中代码，优先向后选择离光标最近的单词，再按一次选中整个单词、行；
ctrl+F12: 浮动窗口显示当前文件结构；
ctrl+shift+N: 查找文件；
ctrl+shift+backspace: 返回到上次编辑的地方；
ctrl+alt+left/right: 前后导航编辑过的地方；
ctrl+alt+up/down: 快速跳转搜索结果；
ctrl+alt+s: 打开设置界面；

shift+shift: 在项目中查找内容；
```

## 三、代码调试
```yml
alt+shift+F9: 选择debug；
alt+shift+F10: 选择run；
ctrl+shift+F9: 编译；
ctrl+shift+F8: 查看断点；

F7: 步入；
F8: 步过；
F9: 恢复程序；

alt+F8: debug时查看选中变量的值；
alt+F9: 运行至光标处；
alt+F10: 定位到断点；

shift+F7: 智能步入；
alt+shift+F7: 强制步入；
shift+F8: 步出；
alt+shift+F8: 强制步出；
```
